So why would I want to donate?



Short answer: Hosting a website is not free

Long answer: This website was created by BYU students who have now graduated. Its purpose was simply to improve how students register for classes -- we've never considered making it a for-profit thing. Once every year a BYU class picks up the project and adds some features to it, but the actual cost of hosting the website has fallen on one of the original creators. And that benevolent BYU grad would like some help covering the hosting costs. If you're a nerdy/techy person and/or aren't intimidated by some acronyms and proper nouns, what we're paying for is an AWS EC2 instance which we use as both a web server and a mongodb database. If you'd like more details beyond that, shoot us an email at support@registerbyu.com.
